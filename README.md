# Professional BTC/USD Trading Signal Agent

Advanced cryptocurrency trading agent with AI-powered analysis running 24/7.

## Features

✅ **Monitor BTC/USD** - Every 30 minutes
✅ **Technical Analysis:**
   - RSI (Relative Strength Index)
   - MACD (Moving Average Convergence Divergence)
   - EMA (Exponential Moving Average) - 20 & 50 period
   - Bollinger Bands

✅ **AI Analysis** - Groq AI (professional-grade)
✅ **Trading Signals:**
   - 🟢 BUY signals
   - 🔴 SELL signals
   - 🟡 HOLD signals

✅ **Risk Management:**
   - Stop Loss (SL) calculation
   - Take Profit (TP) calculation
   - Confidence levels

✅ **24/7 Operation** - Railway cloud
✅ **Telegram Integration** - Real-time alerts

## Tech Stack

- Python 3
- CoinGecko API (data)
- Groq AI (analysis - FREE & UNLIMITED)
- Telegram Bot API
- Railway (hosting)

## Setup

1. Create GitHub repo: `btc-trading-agent`
2. Upload: `professional_btc_agent.py`, `requirements.txt`, `Procfile`
3. Deploy to Railway from GitHub
4. Wait 3-5 minutes
5. Check Telegram for signals every 30 minutes
